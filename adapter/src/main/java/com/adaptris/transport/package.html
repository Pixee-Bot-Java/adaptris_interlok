The Adaptris Transport package contains all the relevant code used to handle a direct connection to a remote machine.

The default implementation is a TCP/IP socket. This can be either a normal plain socket, or a SSL/TLS socket implementation. Further implementations such as a direct MODEM connection, or X.25 connection are available.

Use of the TLS implementation does require JSSE 1.0.2 compliant jars to be available in the classpath. An attempt is made to add the default Sun JSSE 1.0.2 provider - com.sun.net.ssl.internal.ssl.Provider() but no logging of this attempt (or failure) is made. More information about JSSE can be found here

The formal specificiation for TLS can be found on the internet draft RFC2246
